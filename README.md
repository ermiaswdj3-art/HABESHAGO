# 🚖 HABESHAGO

## Project Overview

HABESHAGO is a next-generation Ethiopian ride-hailing and delivery platform designed to provide safe, reliable, affordable, and modern transportation services.

This project is being developed from scratch using professional software engineering practices with a focus on security, scalability, and maintainability.
